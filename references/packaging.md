# Skill Packaging Guide

## Claude Code environment

In Claude Code, place the skill directory under `~/.claude/skills/` and it will be discovered automatically — no packaging needed.

```bash
# Verify the skill is recognized
claude --list-skills
```

## Claude.ai environment (upload .skill file)

A `.skill` file is simply a zip archive with the extension renamed to `.skill`.

### Packaging steps

```bash
# Package a specific skill
cd ~/.claude/skills/
zip -r <skill-name>.skill <skill-name>/
mv <skill-name>.zip <skill-name>.skill
```

Or use the following script (run from the skills directory):

```bash
#!/bin/bash
SKILL_NAME=$1
if [ -z "$SKILL_NAME" ]; then
  echo "Usage: package_skill.sh <skill-name>"
  exit 1
fi

cd ~/.claude/skills/
zip -r "${SKILL_NAME}.skill" "${SKILL_NAME}/"
echo "Generated: ~/.claude/skills/${SKILL_NAME}.skill"
```

### Upload to Claude.ai

1. Open Claude.ai
2. Go to Settings > Skills
3. Click "Upload Skill"
4. Select the `.skill` file
5. The skill is available in conversation immediately after upload

## Batch package all skills generated by self-distill

```bash
cd ~/.claude/skills/
for dir in */; do
  skill_name="${dir%/}"
  zip -r "${skill_name}.skill" "${dir}"
  echo "Packaged: ${skill_name}.skill"
done
```
